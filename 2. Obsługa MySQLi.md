# Obsługa bazy danych w PHP

MySQLi to biblioteka PHP służąca do obsługi baz danych MySQL. Została wprowadzona w PHP 5.0 jako alternatywa dla biblioteki MySQL, aby umożliwić programistom korzystanie z nowszych funkcjonalności MySQL, takich jak przygotowywanie zapytań czy transakcje.

## 🔗 Łączenie się z bazą danych
Aby połączyć się z bazą danych MySQL przy użyciu MySQLi należy zdefiniować skrypt

```php
// TWORZENIE POŁĄCZENIA
$conn = mysqli_connect("localhost", "root", "", NAZWA_BAZY_DANYCH);
```

>[!NOTE]
> Funkcja ``mysqli_connect()`` przyjmuje cztery parametry: ``nazwę hosta``, ``nazwę użytkownika``, ``hasło`` i ``nazwę bazy danych``. Jeśli połączenie zostanie nawiązane pomyślnie, funkcja zwróci ``obiekt połączenia``, który można wykorzystać w dalszej pracy.

Dlatego też potrzebne jest zapisanie go do zmiennej, w tym przypadku `$conn`.

## 📋 Zapytania `SELECT` do bazy danych

>[!NOTE]
> Aby wykonać jakiekolwiek zapytanie do bazy danych należy skorzystać z funkcji `mysqli_query()`, która przyjmuje **dwa argumenty** - **zmienna z połączeniem do bazy danych** i **zapytanie SQL**.

```php
// PRZYKŁADOWE ZAPYTANIE SQL
$sql = "SELECT * FROM users";

// WYKONANIE ZAPYTANIA
$result = mysqli_query($conn, $sql);
```

Natomiast aby zapytanie wybierające do bazy danych można było wyświetlić w naszym pliku z rozszerzeniem `.php`, należy wywołać specjalną funkcję przetwarzającą dane z języka SQL na język PHP - tą funkcją jest `mysqli_fetch_assoc()`, przyjmująca jako argument jedynie wynik z `mysqli_query()`.

```php
// POBIERANIE WYNIKÓW I WYPISYWANIE ICH NA STRONIE
while($row = mysqli_fetch_assoc($result)) {
    echo "<p> Id: " . $row["id"] . " - Imię: " . $row["firstname"] . " - Nazwisko: " . $row["lastname"] . "</p>";
}
```

W pętli `while` przypisujemy do tymczasowej zmiennej `$row` wynik z `mysqli_fetch_assoc()` by wypisać każdy wiersz po koleii.

>[!NOTE]
>Jeżeli mamy pewność, że zwrócony zostanie tylko jeden wiersz, możemy wykorzystać funkcję `mysqli_fetch_row()`, która podobnie do `mysqli_fetch_assoc()` przyjmuje `$result` jako argument.

```php
$data = mysqli_fetch_row($result);
// Ważnym jest aby pamiętać, że zwrócone dane są dalej w tablicy,
// gdzie każdy indeks tej tablicy to poszczególna kolumna (W ZALEŻNOŚCI OD TEGO CO WYBRALIŚMY W SELECT).
echo $data[0]; // id
echo $data[1]; // firstname
echo $data[2]; // lastname
```

>[!TIP]
> Funkcja ``mysqli_num_rows()`` zwraca liczbę wierszy w wyniku zapytania ``SELECT``. Dzięki temu możemy łatwo sprawdzić, ile rekordów zostało zwróconych przez zapytanie.

```php
$sql = "SELECT * FROM users";
$result = mysqli_query($conn, $sql);

$num_rows = mysqli_num_rows($result);
echo "Liczba użytkowników: $num_rows";
```


## ➕ Zapytania `INSERT` do bazy danych

W momencie kiedy chcemy dodać nowy wiersz do tabeli w bazie danych, musimy skorzystać z podobnego przykładu co poprzednio.

```php
$sql = "INSERT INTO users (firstname, lastname) VALUES ('Jan', 'Kowalski')";
mysqli_query($conn, $sql);
```

Jednak rzadko kiedy chcemy dodawać sztywne dane wpisywanie z palca. W tym przypadku aby dane były wprowadzane przez użytkownika, musimy posłużyć się formularzem z pliku `.php`

>[!WARNING]
> Formularz musi zawierać atrybuty
>
> - **action**, który wskazuje który plik ze skryptem PHP ma się wykonać po wysłaniu formularza
> - **method**, który wskazuje jaką metodą mają zostać wysłane dane. Możliwe są `POST` oraz `GET`.

```html
<form action='index.php' method='POST'>
    <label for='imie'>
        <p>Podaj swoje imie:</p>
        <input type='text' name='imie' id='imie'/>
    </label>
    <label for='nazwisko'>
        <p>Podaj swoje nazwisko:</p>
        <input type='text' name='nazwisko' id='nazwisko'/>
    </label>

    <button>Dodaj</button>
</form>
```

Następnie w pliku `index.php` należy zapisać skrypt, który ma zostać wykonany w momencie kliknięcia przycisku `Dodaj` na stronie.


>[!NOTE]
> Przechwytywanie danych z formularza odbywa się za pomocą specjalnej zmiennej globalnej, którą określamy po przez `method` w formularzu. W naszym przypadku będzie to `$_POST`.
>
> Ważne też jest określenie atrybutów `name` w inputach naszego formularza, bo to po nich możemy się odwołać do wartości tych pól np. `$_POST['imie']` będzie przechowywało tekst z inputa tekstowego o `name='imie'`. 

```php
// POBRANIE DANYCH Z FORMULARZA
$imie = $_POST['imie'];
$nazwisko = $_POST['nazwisko'];

// WYKONANIE ZAPYTANIA DO BAZY DANYCH
$sql = "INSERT INTO users (firstname, lastname) VALUES ('$imie', '$nazwisko')";
mysqli_query($conn, $sql);
```

Wystąpi natomiast jeden problem, zaraz po przeładowaniu świeżej strony pojawi się błąd w miejscu skryptu php.

Wszystko to z powodu natychmiastowego wykonania się fragmentu skryptu próbującego pobrać dane z formularza, które nie zostały nawet jeszcze wypełnione. Aby zabezpieczyć się przed tym działaniem należy zapisać prostą instrukcję warunkową z wykorzystaniem globalej zmiennej `$_SERVER['REQUEST_METHOD']`, którą przyrównamy do metody formularza.

```php
// JEŻELI WYSŁANA METODA TO POST TO WYKONAJ KOD WEWNĄTRZ
if($_SERVER['REQUEST_METHOD'] == 'POST'){

// POBRANIE DANYCH Z FORMULARZA
$imie = $_POST['imie'];
$nazwisko = $_POST['nazwisko'];

// WYKONANIE ZAPYTANIA DO BAZY DANYCH
$sql = "INSERT INTO users (firstname, lastname) VALUES ('$imie', '$nazwisko')";
mysqli_query($conn, $sql);
}
```

Jeżeli jednak potrzebowalibyśmy prześledzić nowo dodany wiersz do bazy danych to możemy uzyskać jego id za pomocą funkcji `mysqli_insert_id()`, który przyjmuje połączenie z bazą danych jako argument.
```php
$id = mysqli_insert_id($conn);
echo "Dodano użytkownika o ID: $id";
```

## 🔄 Zapytania `UPDATE` do bazy danych

Może zdarzyć się tak, że będziemy potrzebować wykonać aktualizację w tabeli.

W tym przypadku musimy wykonać ponowne zapytanie `mysqli_query()` jednak samo zapytanie zostanie zmienione na to aktualizujące w języku SQL.

```php
$sql = "UPDATE users SET lastname = 'Nowak' WHERE firstname = 'Jan'";
mysqli_query($conn, $sql);
```

>[!TIP]
> Funkcja ``mysqli_affected_rows()`` zwraca liczbę wierszy dotkniętych ostatnim zapytaniem SQL. Jest to przydatne, gdy chcemy np. sprawdzić, ile rekordów zostało zmienionych w wyniku zapytania ``UPDATE`` lub ``DELETE``.

```php
// JEŻELI WYSŁANA METODA TO POST TO WYKONAJ KOD WEWNĄTRZ
if($_SERVER['REQUEST_METHOD'] == 'POST'){

// POBRANIE DANYCH Z FORMULARZA
$nazwisko = $_POST['nazwisko'];

// WYKONANIE ZAPYTANIA DO BAZY DANYCH
$sql = "UPDATE users SET lastname = '$nazwisko' WHERE firstname = 'Jan'";
mysqli_query($conn, $sql);

$ile_wierszy = mysqli_affected_rows($conn);
if($ile_wierszy > 0){
    echo "Zaktualizowano " . $ile_wierszy . " wierszy.";
}
}
```

## 🗑️ Zapytania `DELETE` do bazy danych
Gdy chcemy usunąć wiersz lub wiele wierszy w naszej bazie danych należy zastosować odpowiednio zmodyfikowany kod z poprzedniego listingu.

>[!TIP]
> Jeżeli mamy kilka formularzy wykonujących metodę `POST` możemy dodać jeszcze jeden warunek sprawdzający czy został "ustawiony" przycisk danego formularza, któremu również możemy nadać atrybut `name`.
>
> Funkcja `isset()` sprawdza czy dany element formularza HTML został ustawiony.

```php
// JEŻELI WYSŁANA METODA TO POST TO WYKONAJ KOD WEWNĄTRZ
if($_SERVER['REQUEST_METHOD'] == 'POST' && isset($_POST['remove_btn']){

// POBRANIE DANYCH Z FORMULARZA
$id = $_POST['id'];

// WYKONANIE ZAPYTANIA DO BAZY DANYCH
$sql = "DELETE FROM users WHERE id = $id";
mysqli_query($conn, $sql);

$ile_wierszy = mysqli_affected_rows($conn);
if($ile_wierszy > 0){
    echo "Usunięto " . $ile_wierszy . " wierszy.";
}
}
```

### 🌠 Zadania dotyczące MySQLi w PHP

1. **Połączenie z bazą danych**
- Napisz skrypt PHP, który nawiązuje połączenie z bazą danych `php_przyklad`.
- Jeśli połączenie się powiedzie, wyświetl komunikat: `Połączono z bazą danych`.
- Jeśli połączenie się nie powiedzie, wyświetl komunikat o błędzie.

**Podpowiedź:**
Użyj funkcji `mysqli_connect(host, user, password, database)`, a następnie sprawdź, czy połączenie się udało.

2. **Wyświetlenie listy użytkowników**
- Napisz skrypt PHP, który pobiera wszystkich użytkowników z tabeli `uzytkownicy`.
- Wyniki wyświetl w tabeli HTML z kolumnami: `ID`, `Nazwa`, `E-mail`.

**Podpowiedź:**
Użyj zapytania `SELECT * FROM uzytkownicy`, a następnie `mysqli_fetch_assoc()` do pobrania danych.

3. **Dodanie nowego użytkownika**
- Stwórz formularz HTML, w którym użytkownik poda swoją nazwę, e-mail oraz hasło.
- Napisz skrypt PHP, który po wysłaniu formularza doda nowego użytkownika do tabeli `uzytkownicy`.
- Po pomyślnym dodaniu użytkownika wyświetl komunikat `Użytkownik dodany pomyślnie`.

**Podpowiedź:**
Użyj zapytania `INSERT INTO uzytkownicy (nazwa, email, haslo) VALUES (...)`.

4. **Aktualizacja danych użytkownika**
- Stwórz formularz HTML, w którym użytkownik poda swoje `ID`, nową nazwę i nowy e-mail.
- Po wysłaniu formularza, skrypt PHP powinien zaktualizować dane w bazie.

**Podpowiedź:**
Użyj zapytania `UPDATE uzytkownicy SET nazwa = ..., email = ... WHERE id = ...`.

5. **Usunięcie użytkownika**
- Stwórz formularz HTML, w którym użytkownik poda `ID` użytkownika do usunięcia.
- Po wysłaniu formularza skrypt PHP powinien usunąć użytkownika z tabeli.

**Podpowiedź:**
Użyj `DELETE FROM uzytkownicy WHERE id = ...`.

6. **Lista dostępnych produktów**
- Napisz skrypt PHP, który pobiera i wyświetla tylko te produkty, które mają `dostępność = 1`.
- Wyniki wyświetl w tabeli HTML z kolumnami: `ID`, `Nazwa`, `Cena`.

**Podpowiedź:**
Użyj `SELECT * FROM produkty WHERE dostępność = 1`.

7. **Dodanie nowego produktu**
- Stwórz formularz HTML, w którym użytkownik poda nazwę produktu, opis, cenę i kategorię.
- Skrypt PHP powinien dodać produkt do tabeli `produkty`.

**Podpowiedź:**
Użyj `INSERT INTO produkty (nazwa, opis, cena, kategoria) VALUES (...)`.

8. Zaktualizowanie ceny produktu
- Stwórz formularz HTML, w którym użytkownik poda `ID` produktu oraz nową cenę.
- Po wysłaniu formularza, skrypt PHP powinien zaktualizować cenę w bazie danych.

**Podpowiedź:**
Użyj `UPDATE produkty SET cena = ... WHERE id = ...`.

9. **Lista produktów w koszyku użytkownika**
- Napisz skrypt PHP, który pobiera produkty znajdujące się w koszyku danego użytkownika (na podstawie jego `ID`).
- Wyświetl nazwę produktu, jego cenę i kategorię.

**Podpowiedź:**
Połącz tabele `koszyk` i `produkty` za pomocą `JOIN` w zapytaniu SQL.

10. **Usunięcie produktu z koszyka**
- Stwórz formularz HTML, w którym użytkownik poda `ID` rekordu z koszyka do usunięcia.
- Po wysłaniu formularza skrypt PHP powinien usunąć ten wpis.

**Podpowiedź:**
Użyj `DELETE FROM koszyk WHERE id_koszyka = ...`.



## 🔒 Zapytania parametryzowane do bazy danych

>[!CAUTION]
> Funkcje z biblioteki MySQLi pozwalają na wykonywanie zapytań SQL do bazy danych oraz na efektywne zarządzanie połączeniem z bazą danych. Aby korzystać z tych funkcji, należy najpierw nawiązać połączenie z bazą danych za pomocą funkcji mysqli_connect(). Następnie można wykonywać zapytania SQL, korzystając z funkcji takich jak mysqli_query(), mysqli_prepare(), mysqli_stmt_execute() oraz mysqli_fetch_assoc(). Ważne jest również pamiętanie o poprawnym sanitizingu wartości parametrów oraz o zabezpieczeniu przed atakami typu SQL injection poprzez użycie prepared statements.

Przygotowanie zapytania w MySQL polega na zastępywaniu wartości parametrów znakiem zapytania "?".

```php
$sql = "SELECT * FROM users WHERE id = ? AND lastname = ?";
```

Aby przygotować zmienną typu ``statement``, używając funkcji ``mysqli_prepare()``, która przyjmuje dwa parametry: ``obiekt połączenia`` i ``zapytanie SQL``.

```php
$stmt = mysqli_prepare($conn, $sql);
```

Następnie należy przypisać wartości parametrów do zmiennej typu ``statement``, używając funkcji ``mysqli_stmt_bind_param()``, która przyjmuje jako pierwszy parametr zmienną typu ``statement``, a następnie ``listę wartości parametrów``. W przypadku zapytania SQL z dwoma parametrami typu integer, należy użyć typu ``i``.

```php
$id = 1;
$lastname = "Kowalski";
mysqli_stmt_bind_param($stmt, "is", $id, $lastname);
```

Aby wykonać zapytanie, należy użyć funkcji ``mysqli_stmt_execute()``, która przyjmuje jako parametr zmienną typu ``statement``.

```php
mysqli_stmt_execute($stmt);
```

Ostatecznie pobranie wyników zapytania, dokonuje się używając funkcji ``mysqli_stmt_get_result()``, która przyjmuje jako parametr zmienną typu ``statement``.

```php
$result = mysqli_stmt_get_result($stmt);
```

Przetworzenie wyniku zapytania, używając funkcji ``mysqli_fetch_assoc()`` lub ``mysqli_fetch_array()``.
Pierwsza funkcja zwróci tablice asocjacyjną, druga zwykłą.

```php
while($row = mysqli_fetch_assoc($result)) {
    echo "Id: " . $row["id"] . " - Imię: " . $row["firstname"] . " - Nazwisko: " . $row["lastname"] . "<br>";
}
```

Należałoby również zwolnić zasoby, używając funkcji ``mysqli_free_result()`` i ``mysqli_stmt_close()``.

```php
mysqli_free_result($result);
mysqli_stmt_close($stmt);
```

Całość powinna wyglądać mniej wiecej tak:

```php
$sql = "SELECT firstname, lastname FROM users WHERE id = ?";
$stmt = mysqli_prepare($conn, $sql);
$id = 1;
mysqli_stmt_bind_param($stmt, "i", $id);
mysqli_stmt_execute($stmt);
mysqli_stmt_bind_result($stmt, $firstname, $lastname);
mysqli_stmt_fetch($stmt);
echo "Imię: " . $firstname . " - Nazwisko: " . $lastname . "<br>";
mysqli_stmt_close($stmt);
```

W pierwszej funkcji, czyli ``mysqli_stmt_bind_param()``, przekazujemy do funkcji argumenty w kolejności, w jakiej są użyte w zapytaniu SQL. Następnie musimy określić ich typy, które zależą od typów wartości w bazie danych oraz od typów PHP. Na końcu przypisujemy do tych argumentów wartości.

Kolejna funkcja, ``mysqli_stmt_execute()``, wywołuje zapytanie SQL z użyciem przypisanych wartości z poprzedniej funkcji.

Funkcja ``mysqli_fetch_assoc()`` pobiera kolejne wiersze z wyniku zapytania i zwraca je jako asocjacyjną tablicę, gdzie kluczami są nazwy kolumn, a wartościami wartości wierszy.

Funkcja ``mysqli_stmt_bind_result()`` pozwala na przypisanie wyniku zapytania SQL do zmiennych. Przy użyciu tej funkcji możemy pobrać wyniki zapytania w formie zmiennych, co umożliwia łatwiejszą dalszą obróbkę wyników.
